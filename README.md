# NovaStore SQL Project

This project is a relational database design and SQL querying project developed using SQL Server.

## Project Scope
- Designing a relational database schema
- Creating tables with primary and foreign keys
- Inserting sample (dummy) data
- Writing SQL queries for analysis and reporting
- Creating a database diagram (ER Diagram)

## Database Structure
The database consists of the following tables:
- Categories
- Products
- Customers
- Orders
- OrderDetails

These tables are connected using primary key and foreign key relationships.

## Features & Queries
The project includes various SQL queries such as:
- Listing products with low stock
- Joining customers and orders
- Detailed product listing for a specific customer
- Product count by category
- Total spending per customer
- Date difference calculation for orders

## Technologies Used
- SQL Server
- T-SQL
